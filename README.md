## 1.top
#### top 명령어는 현재 **OS의 메모리 사용량**, **CPU 사용량** 등을 실시간으로 갱신되는 값으로 나타내주는 명령어로
#### 프로그램 구동시에 서버의 부하가 어느정도 되는지 테스트나 모니터링 할 때 주요하게 사용한다.
#### top 명령어의 옵션은 다음과 같이 정리할 수 있다.
| 명령어 | 기능 | 문법 |
|:--------:|:-------|:-------|
| **-d** | 업데이트 간격(단위: 초) 지정 | `top -d` |
| **-p** | 특정 PID만 모니터링 | `top -p PIDNUM` |
| **-n** | 실행 횟수 지정 | `top -n` |
